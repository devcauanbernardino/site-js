O que é JavaScript?

JavaScript (JS) é uma linguagem de programação de alto nível, interpretada e orientada a objetos baseada em protótipos, criada inicialmente para rodar no navegador e permitir interatividade em páginas web.

História

JavaScript foi criado em 1995 por Brendan Eich na Netscape para tornar as páginas web mais interativas. Inicialmente chamado de Mocha e depois LiveScript, recebeu o nome JavaScript para aproveitar a popularidade do Java, embora as duas linguagens sejam diferentes.

Em 1997, a ECMA International padronizou a linguagem como ECMAScript, garantindo compatibilidade entre navegadores. Nos anos 2000, bibliotecas como jQuery facilitaram o uso da linguagem e a manipulação do DOM.

Em 2009, o Node.js permitiu usar JavaScript no lado do servidor, e em 2015 o ECMAScript 6 trouxe recursos modernos como let, const, arrow functions, classes e promises, tornando a linguagem mais poderosa.

Hoje, JavaScript é a linguagem mais usada na web, base de frameworks como React, Angular e Vue, e também utilizada em back-end, aplicativos móveis, jogos e até soluções de inteligência artificial.
